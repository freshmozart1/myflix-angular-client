UserService

Description

This service is responsible for managing user data, including login, registration, and user updates.

Param: fetchApiData

Service to interact with the backend API.

Description

Getter and setter for the user object. The setter handles user login and logout, while the getter retrieves the user data from local storage.

Returns

The user object containing username and token.

Optional Angular Material snack bar to show notifications.
